# 연구실 안전관리 시스템 Frontend

## 실행 방법

```bash
npm install
npm run dev
```

기본 API 주소는 `http://localhost:8080`입니다. 변경하려면 `.env`에 아래 값을 넣으세요.

```env
VITE_API_BASE_URL=http://localhost:8080
```

## 반영 내용

- 회원가입: 학과/부서 단일 선택, 관리 부서 선택 추가
- 로그인: 예시 아이디/비밀번호 문구 제거
- 사용자 관리: 상태/권한/학과/부서 필터, 검색, 한글 라벨 표시
- 연구실 관리: 목록은 연구실명/학과/책임자만 간단 표시, 검색/필터, 상세 조회 분리
- 화학물질 관리: 등록은 ADMIN만 표시, 조회는 로그인 사용자 가능, 검색/필터, 연구실 자동완성 선택
- 폐기물 관리: 검색/필터, 목록 요약, 담당자 입력 제거, 로그인 사용자가 담당자로 저장되도록 요청
- 점검 양식: 검색/필터, 목록에서는 파일 숨김, 등록은 SAFETY_MANAGER/ADMIN만 표시
- 안전교육: 검색/필터, 목록에서는 영상/파일 숨김, 등록은 EDUCATION_MANAGER/ADMIN만 표시
